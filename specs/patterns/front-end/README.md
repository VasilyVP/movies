# Front-End Patterns

This folder stores reusable front-end implementation patterns used in this project.

## Available Patterns

- custom_http_hook.md
  - Reusable pattern for a typed HTTP hook built with TanStack Query.
  - Use when multiple components need the same fetch, loading, cache, and error behavior.

## How To Use

1. Copy the pattern into a feature-specific spec.
2. Replace placeholders with feature names, endpoint paths, and response types.
3. Keep acceptance criteria section and adapt it to the feature behavior.
